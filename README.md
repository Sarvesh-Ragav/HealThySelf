# HealThySelf
Overview
This project is developed for a hackathon event, aiming to provide a personalized health and wellness platform that offers both physical and mental health support. Users can calculate BMI, receive workout plans, track calorie intake, and access mental health resources like music therapy and a coping companion chatbot.
